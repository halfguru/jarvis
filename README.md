## JARVIS

Just A Rather Very Intelligent System

### Prerequisites

Python3 & pip
```
python -m pip install -U pip
pip install google-cloud
pip install pyttsx3
pip install webbrowser
pip install wikipedia
pip install SpeechRecognition
pip install bs4
```
